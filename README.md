# EEL-Project-Design-PBL
This project is situated at the critical intersection of technology and education. By applying User-Centred Design (UCD) principles and leveraging the professional design tool Figma, this project addresses the usability gaps in an existing online learning platform called 'Crossing  Knowledge.'
